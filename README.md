# Tech Challenge — Olist

## Objetivo

Investigar como o desempenho logístico influencia a satisfação dos clientes e o potencial de retenção e crescimento das operações de comércio eletrônico.

## Pergunta Norteadora

O desempenho logístico impacta diretamente a satisfação dos clientes e o potencial de retenção e crescimento das operações de comércio eletrônico?

## Dataset

Brazilian E-commerce Dataset disponibilizado pela Olist.

## Principais Indicadores

- Receita
- Quantidade de pedidos
- Tempo médio de aprovação
- Tempo médio de entrega
- Taxa de entregas no prazo
- Avaliação média dos clientes
- Retenção de clientes
- Ticket médio

## Principais Resultados

- Atrasos impactam negativamente a satisfação.
- Entregas mais rápidas apresentam avaliações superiores.
- O crescimento da receita ocorreu principalmente pela aquisição de novos clientes.
- A taxa de recompra mostrou-se baixa.
- A logística se apresenta como fator estratégico para retenção e crescimento.
